# Test Suite

Comprehensive test suite for the blog website.

## Structure

- `unit/` - Unit tests for individual modules
- `integration/` - Integration tests for component interactions
- `e2e/` - End-to-end tests for full user flows
- `visual/` - Visual regression tests

## Running Tests

Tests are designed to run in the browser. Open the test HTML files in a browser or use a test runner.

## Test Coverage

- Blog engine (post loading, rendering, TOC generation)
- Template engine (rendering, conditionals)
- Markdown loader
- Navigation (keyboard shortcuts, menu)
- Styling (dark/light themes, responsive design)
- Accessibility (ARIA labels, keyboard navigation)
