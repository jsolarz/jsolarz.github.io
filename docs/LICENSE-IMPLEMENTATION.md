## Alternative Options

If you want different protection levels:

### More Restrictive
- **All Rights Reserved** - Maximum protection
- **CC BY-NC-ND** (current) - No commercial, no derivatives
- **CC BY-ND** - Commercial OK, no derivatives

### Less Restrictive
- **CC BY-NC-SA** - No commercial, allows derivatives
- **CC BY-SA** - Commercial OK, allows derivatives
- **CC BY** - Attribution only
